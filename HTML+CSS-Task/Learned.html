- ما هو الإنترنت؟ هو شبكة اتصال عبارة عن أسلاك و... (ما لحقت أكتب) نوصفه بأنه البنية التحتية (الطريق) بدون وجوده ماكو ويبسايت - ما هو الويب؟ هو مجموعة مواقع تعرض عبر الإنترنت له إصدارات، كل إصدار أحدث من الآخر وفيه إمكانيات أكثر وله عدة طبقات الي هي: طبقة السطح، الديب ويب (هو الدومينات خارج نطاق www)، الدارك ويب (هو الذي يكون فيه أشياء غير قانونية) - ما هي السيرفرات؟ هي عبارة عن أجهزة وأي جهاز عنده القدرة على الإرسال والاستلام، ويكون له غرض معين لتقديم خدمة - ما هو العنوان الافتراضي؟ هو العنوان لأي جهاز متصل بالشبكة IP Address مكون من 4 خانات، كل خانة فيها رقم بين 0-255 - ماذا عن العنوان الفيزيائي؟ عندك IP Adress وMAC Address الأول متغير والثاني غير قابل للتغيير داخل الشبكة (جماعة الأمن السيبراني عندهم حالات خاصة يعدلون على الماك) والثاني هو عنوان الجهاز من التصنيع - ما هو الـDNS (Domain Naming System)؟ هي عبارة عن سيرفرات تخزن كل عنوان IP لكل موقع واسمه Domain name موجود 4 سيرفرات متوزعة حول العالم: آسياـ أفريقيا، أوروبا، والأمريكتين من خلال المتصفح حتى نرسل طلب لموقع معين، إما نبعث الدومين للـDNS ويرجعلنا الـIP ونستخدمه لأخذ البيانات من الـWeb Server - ما هو الدومين؟ هو العنوان الافتراضي الذي يحتوي Domain name وSub Domain وهكذا كله يكون URL - ما هو الـURL؟ protocol بروتوكول https:// sub domain دومين فرعي video. أو www. domain name عنوان الدومين google. وفيه top level domain مثل co.uk أو .com وهكذا port البورت المعتمد من السيرفر :80 (المكان الي نتصل من خلاله بالسيرفر) path المسار /videoplay query الاستعلام ? parameters معلمات dicid=-724692761&hl=en الي هي متغيرات مثلا وقت الفيديو باليوتيوب fragment فراكمينت #00h02m30s جزئية ممكن نستخدمها كمبرمجين حتى ما تأثر على الـURL - قواعد الويب (البروتوكولات) الأكثر استخداما: TCP/IP: هي نوع من البروتوكولات وكل وحدة منها عندها 5 طبقات من كل طرف سواء المستقبل أو المستلم وهي أحدث شي وأكثر شي مستخدمة الطبقة الأولى طبقة الشبكة Network وهي بيها بكتات وهي باينري ويتم تغليفها وتمريرها عبر الطبقات حتى تصل طبقة الـApplication ولما نصل للمستلم نرجع بالعكس ويصير بالنهاية Handshake هو وين موجود؟ بطبقة الشبكة Network layer - SMTP مال إيميلات - FTP و SFTP نقل ملفات واحد مؤمن والثاني لا - PPP هي مال Puplic key وتشفير المحادثة من الطرفين HTTP, HTTPS, , , , , - محركات البحث كيف تعمل محركات البحث؟ أولا Crawling بوتات غوغل تعمل crawl للويب حتى تلاقي الصفحات الجديدة أو المحتوى المحدث. ثم Indexing يتم التعرف على المعلومات وترتيبها بواسطة البوتات ثم Ranking - المتصفحات هي عبارة عن برامج تعتبر واجهتنا للويب الدوال GET, POST, PUT, DELETE فقط مسميات لكن حسب برمجتنا للباك كود الحالة Status Code: 1XX – INFORMATIONAL 2XX – SUCCESS 3XX – REDIRECTION 4XX – CLIENT ERROR 5XX – SERVER ERROR - الهيدرز Headers هو المكون الرئيسي الي يوضح لي الطلب وتفاصيله وحالته وكذا - الكوكيز خزن دائم مو مؤقت - جيسون عبارة عن ملفات تحتوي قيم بشكل key وvalue الفرونت والباك؟ مقدمة إلى الباك-إند: يستلم طلبات ويعالجها ويرجع الرد - أنواع الـAPI 1. Restfull API: يرجع داتا كاملة كامل كستندر وممكن يرجع شي محدد حسب برمجتنا 2. GraphQL يعطي شيء محدد حسب الطلب ويوفر ذاكرة 3. SOAP يطلب تفاصيل كاملة عند الطلب (يفيد عند احتياج الأمان) 4. gRPC أكثر واحد سريع - ما هي قواعد البيانات DataBase؟ هي مخزن البيانات مالتنا وكل مجموعة بيانات محطوطة بمكان محدد (جدول) - أنواع قواعد البيانات: 1. العلائقية SQL 2. اللا علائقية NoSQL 3. المعتمدة على الأوبجت 4. المخطط - ORM هي ترجمة لغة البرمجة مال باك إلى SQL Query
